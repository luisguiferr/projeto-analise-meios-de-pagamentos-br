# projeto-analise-meios-de-pagamentos-br
Análise dos meios de pagamento no Brasil (2010–2025) utilizando dados da API Olinda do Banco Central. Projeto em Python/Jupyter Notebook com foco em valores transacionados, quantidade de operações e impacto do Pix. Inclui tratamento, visualizações e evolução de TED, Pix, DOC, TEC, boleto e cheque.
